# 🎬 Movie Website

A movie browsing website showcasing popular Malayalam cinema directors and their films, built with HTML, CSS, and Bootstrap.

## 📌 Features

- Responsive navbar with search and login fields
- Dropdown menus for Genres and Directors
- Bootstrap carousel highlighting featured directors
- Movie sections organized by director:
  - Joshiy
  - Ranjith
  - Vineeth Sreenivasan
  - Lijo Jose Pellissery
- Hover zoom effect on movie cards
- Feedback form in the footer

## 🛠️ Built With

- HTML5
- CSS3
- [Bootstrap 5.3.7](https://getbootstrap.com/)
- [AOS (Animate On Scroll)](https://michalsnik.github.io/aos/)
- [Font Awesome](https://fontawesome.com/)

## 🚀 Getting Started

1. Clone this repository
```bash
   git clone https://github.com/ADSN098/Movie-Website.git
```
2. Open the project folder
3. Run it using a local server (recommended), for example:
```bash
   npx serve .
```
4. Open `http://localhost:3000` in your browser
